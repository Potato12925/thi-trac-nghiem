# JSP Rules

- Use JSP + JSTL only
- Use EL: ${}
- Use Bootstrap
- Use Spring Form Tag Library
- Use root theme.css for color
  Avoid:

- Java scriptlets <% %>
- Business logic in JSP

Location:

- /WEB-INF/views/

Reusable layouts:

- /WEB-INF/views/Shared/

them.css:

- src/main/webapp/assets/css/theme.css
